PT-BR

Projeto acadêmico de Python usando Selenium para automatização de emissão de notas fiscais de serviço usando como exemplo um site ficticio de prefeitura.

Automatização feita usando bibliotecas Selenium e OS.

Utilizado Selenium / webdriver para localizar os objetos na página web e enviar os textos preenchidos. Os dados dos clientes estão armazenados no excel "NotasEmitir". Ao final faz o donwload da nota fiscal emitida.

EN-US

Academic project using Python focused in Selenium library to automate the creation of invoices. This project uses a ficticious website which invoices are created.

The automation uses libraries such as Selenium and OS.

Selenium / webdriver locates the objects in the website using it's XPATH or class_name. 
The information of the custumers were stored in an excel file named "NotasEmitir". After all data is filled the script donwloads the invoice.
